Guía Completa de Etiquetas HTML con Ejemplos

Este documento explica el uso de las etiquetas HTML más comunes con ejemplos claros.

Texto y Párrafos

<h1>, <h2>, <h3> - Títulos

Sirven para definir títulos y subtítulos. <h1> es el más importante y <h3> es de menor rango.

Ejemplo:

Título Principal del Artículo

Un subtítulo importante

Un punto específico dentro del subtítulo

<p> - Párrafo

Crea un bloque de texto separado, es decir, un párrafo.

Este es un párrafo de texto que explica un concepto. Es la etiqueta más común para escribir texto.

<strong> y <b> - Negrita

<strong> se usa para indicar importancia; <b> solo aplica negrita visual.

<em> y <i> - Cursiva

<em> da énfasis. <i> se usa para palabras extranjeras o técnicas.

Saltos y Separadores

<br> - Salto de línea

Ejemplo:

Calle de la Alegría, 24
28012, Madrid
España

<hr> - Línea horizontal

Sirve para separar secciones de contenido.

Citas

<blockquote> - Cita larga

La imaginación es más importante que el conocimiento. El conocimiento es limitado, mientras que la imaginación abarca el mundo entero.

<q> - Cita corta

Como dijo mi abuelo, nunca es tarde si la dicha es buena.

<cite> - Título de una obra

Mi libro favorito es El Señor de los Anillos.

Listas

<ul> y <ol>

<ul> es desordenada y <ol> ordenada.

Ejemplo de lista desordenada:

  • Leche
  • Pan
  • Manzanas

Ejemplo de lista ordenada:

  1. Abrir la puerta
  2. Encender la luz
  3. Cerrar la puerta

Formato de Texto Adicional

<abbr> - Abreviaturas

Ejemplo: OMS

<u>, <del>, <ins>, <s>

Mi fruta favorita es la pera manzana.

<code> y <pre> - Código

Ejemplo corto: console.log("Hola Mundo")

function saludar() {
    console.log("¡Hola, mundo!");
}

Estructura Semántica en HTML5

HTML5 incorporó nuevas etiquetas que permiten organizar el contenido con significado, facilitando la lectura por buscadores y lectores de pantalla.

Estructura básica de una página moderna

     → Cabecera del sitio o de una sección

        → Menú o enlaces de navegación

       → Contenido principal de la página

    → Artículo o bloque de contenido independiente

    → Sección temática o agrupación de contenido

      → Información complementaria o lateral

     → Pie de página con créditos o enlaces secundarios

Explicación de las etiquetas semánticas
